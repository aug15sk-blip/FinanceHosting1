# Personal Finance Dashboard

A beautiful, feature-rich personal finance dashboard built with vanilla HTML, CSS, and JavaScript.

## Features

- **Expense Tracking**: Add, edit, and delete expenses with categories, dates, descriptions, and tags
- **Budget Management**: Set monthly and yearly budgets per category with visual progress tracking
- **Interactive Charts**: 
  - Category breakdown (doughnut chart)
  - Monthly comparison (bar chart)
  - Yearly overview (bar chart)
  - Spending trends (line chart)
- **Advanced Filtering**: Filter by date range, category, search, and sort by various criteria
- **Real-time Statistics**: 
  - Total expenses
  - Remaining budget
  - Average daily spending
  - Active categories count
- **Budget Progress Bars**: Visual indicators showing budget utilization with color-coded warnings
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Local Storage**: All data is saved locally in your browser

## How to Use

1. Open `index.html` in a web browser
2. Click "Add Expense" to add your first expense
3. Set budgets by clicking "Set Budget" for each category
4. Use filters to view specific time periods or categories
5. View your spending patterns through interactive charts

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (ES6+)
- Chart.js (via CDN) for visualizations
- LocalStorage API for data persistence

## Browser Support

Works in all modern browsers that support:
- ES6+ JavaScript
- CSS Grid and Flexbox
- LocalStorage API
- Chart.js library

## Data Storage

All data is stored locally in your browser's localStorage. Your data will persist between sessions but will be cleared if you clear your browser data.
